# WATS/CEWA 6155 Course Overview

{/* ![WATS 5150 Canvas](/images/pics/WATS-5150_Canvas.png) */}


Welcome to the Fluvial Geomorphology - Reading Riverscapes Course Website!

This online resource is designed primarily for students enrolled in the online fluvial geomorphology courses, including [WATS 6900/6155](/syllabus/),  and [CEWA 6900/6155](/syllabus/) taught through [Utah State University](https://qcnr.usu.edu/wats/). This is no longer the course page for WATS/GEO 5150/6150 However, it also serves as a valuable introduction to the fascinating field of fluvial geomorphology and will build an appreciation of riverscapes for students and practitioners seeking knowledge in this area.
